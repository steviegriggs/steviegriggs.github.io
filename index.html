Latitude vs. Weather Analysis Project Overview:

If you have ever wondered how the weather changes in relation to latitude, then this is the place for you. This project reviews how temperature, humidity, wind speed, and cloudiness change in relation to latitude. The data used in the analysis was pulled from the OpenWeatherMap API and the plots were created using Matplotlib.
